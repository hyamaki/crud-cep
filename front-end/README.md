# Teste Qentrega

Teste de CRUD de clientes

## Pré-requisitos

- Node.js instalado
- API backend rodando em `http://localhost:4000`

## Como executar o projeto

1. Instale as dependências:

```bash
npm install
```

2. Inicie o projeto:

```bash
npm run start
```

O projeto será executado em [http://localhost:3000](http://localhost:3000).
